# Chrome Extension: Academic Paper Summarizer and Citation Extractor

A Chrome extension for researchers, students, and academics that generates structured summaries of research papers on arXiv, Google Scholar, PubMed, and JSTOR. Extracts methodology, key findings, and limitations into a structured abstract. Automatically formats citations in APA, MLA, and Chicago style for one-click copying. Tracks a personal reading list with notes and highlights, and exports annotated reading lists as formatted bibliographies.

## Installation

### Chrome
1. Download and extract the ZIP
2. Go to `chrome://extensions/`
3. Enable "Developer mode"
4. Click "Load unpacked" and select the extracted folder

### Firefox
1. Download the Firefox version
2. Go to `about:debugging`
3. Click "Load Temporary Add-on"
